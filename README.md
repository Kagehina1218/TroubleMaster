# Pet Detective Game (Trouble Master)
>By: Grace Li (kagehina1218), Emily Lu (emily0107)

The project goal is to develope a 2D mini game on a computer that has elements on pets (mainly cats) and a little bit of detective / puzzle solving, 
where the player would be a pet owner who has to figure out which pet is causing which trouble in the house each day. 
(Ex: There might be scratches on the sofa that’s caused by one of the cats, some broken pieces of vase on the floor damaged by a dog…).
  
## Requirements  
Python 3.13.2  
Python **Pygame** library  
```bash
pip install pygame
```

## Installation Instructions  
1. Download python from [Python.org](https://www.python.org/downloads/)  
   - After installation, you can check python version by using command: `python --version`  
2. Clone the repository by using command: `git clone https://github.com/Kagehina1218/Spring2025_CS122.git`  
  
## Usage Instructions  
After cloned the repository to your local machine  
You can then run `main.py` and enjoy the game play!  
   - The `main.py` file is can be find within `pygame` folder > `code` folder > `main.py`
